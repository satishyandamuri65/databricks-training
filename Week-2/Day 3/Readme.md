# Week 2 - Day 3

## Overview

Day 3 practiced SQL on previous year Cognizant hiring questions. The questions are based on two databases — College Management System and Car Rental System. Queries focused on joins, string functions, subqueries, DDL and DML operations.

---

## Database 1 — College Management System

**Tables:** Department, Staff, Student, Subject, Mark

### Topics Covered

- CONCAT to display address and city of students
- Filtering departments by block number using WHERE
- INSERT records into Department table
- INNER JOIN between Student and Department filtered by city
- ORDER BY on joined results

---

## Database 2 — Car Rental System

**Tables:** owners, customers, cars, rentals

### Topics Covered

- Filtering cars by car type using IN
- Filtering Maruthi Sedan cars using LIKE and AND
- ALTER TABLE to add a new column
- CREATE TABLE for owners
- INSERT records into rentals
- Subquery to find cars not taken for rent using NOT IN
- CONCAT to generate username and password for owners
- Subquery with DISTINCT to find Maruthi car owners
- COUNT with GROUP BY to find number of trips per car
- Date filtering using MONTH and YEAR functions
