# Week 2 - Day 5

## Overview

Day 5 practiced SQL on previous year Cognizant hiring questions. The questions are based on three databases — Hospital, Bus Booking System, and a Random/Mixed database. Covered joins, subqueries, string functions, date filtering, and password generation logic.

---

## Database 5 — Hospital

**Tables:** PATIENT, APPOINTMENT

### Topics Covered

- DISTINCT with implicit join to display patient details for June 2019 appointments
- INNER JOIN between PATIENT and APPOINTMENT filtered by appointment reason
- Date filtering using MONTH and YEAR on appointment date

---

## Database 6 — Bus Booking System

**Tables:** buses, schedule, users, bookingdetails

### Topics Covered

- Self join on schedule to find buses with swapped source and destination
- Subquery with NOT IN to find users who never used HDFC bank
- SUBSTR to generate password from username and phone number
- CONCAT to generate USERNAME and PASSWORD for customers
